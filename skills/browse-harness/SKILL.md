# Browse Harness — 하네스 카탈로그 검색

100개 도메인 하네스를 카테고리별 또는 키워드로 검색한다.

## 사용법

- `/browse-harness` — 전체 카테고리 목록
- `/browse-harness 콘텐츠` — 콘���츠 카테고리의 하네스 목록
- `/browse-harness 유튜브` — 키워드 검색

## 카탈로그 소스

`harnesses/INDEX.md`를 읽어 하네스 목록을 표시한다.

## 표시 형식

### 카테고리 목록 (인자 없을 때)

```
## 하네스 카탈로그 (100개)

| 카테고리 | 수량 | 설명 |
|----------|------|------|
| 콘텐츠 | 15 | YouTube, 팟캐스트, 뉴스레터, 게임 내러티브 등 |
| 소프트웨어 | 15 | 풀스택 웹앱, API, 모바일, DevOps 등 |
| 데이터/ML | 12 | ML 실험, 데이터 분석, NLP 등 |
| 비즈니스 | 13 | 스타트업, 시장 조사, 가격 전략 등 |
| 교육 | 10 | 언어, 시험, 코딩, 토론 시뮬레이터 등 |
| 법률 | 7 | 계약, 특허, GDPR, 규정 등 |
| 건강 | 8 | 식단, 피트니스, 세금, 여행 등 |
| 커뮤니케이션 | 8 | 기술 문서, 제안서, 연설문 등 |
| 운영 | 7 | 채용, 온보딩, 프로세스 감사 등 |
| 전문 | 5 | 부동산, 이커머스, 도시계획 등 |

카테고리 이름을 입력하면 상세 목록을 보여드립니다.
예: `/browse-harness 콘텐츠`
```

### 카테고리 상세 (카테고리 지정 시)

```
## 콘텐츠 하네스 (15개)

| 하네스 | 에이전트 | 설명 |
|--------|----------|------|
| youtube-production | 5명 | 영상 기획, 대본, 썸네일, SEO |
| podcast-studio | 5명 | 팟캐스트 기획, 대본, 편집 가이드 |
| newsletter-engine | 5명 | 뉴스레터 기획, 작성, 배포 |
| ... | ... | ... |

하네스 이름을 지정하면 상세 정보를 보여드립니다.
- `/use-harness youtube-production` — 바로 실행
- `/browse-harness youtube-production` — 상세 보기
```

### 하네스 상세 (특정 하네스 지정 시)

해당 하네스의 `CLAUDE.md`를 읽어 표시:
- 에이전트 목록과 역할
- 스킬 목록
- 워크플로우 (단계별 진행)
- 산출물 목록

## 키워드 검색

키워드가 카테고리명이 아니면 INDEX.md에서 하네스 이름과 설명을 검색한다.
매칭되는 하네스가 있으면 목록으로 표시, 없으면 가장 유사한 것을 제안한다.
