# Skip — 태스크 건너뛰기

특정 태스크를 건너뛰고 다음으로 넘어간다.

## 동작

1. 대상 태스크 확인:
   - `/skip` → 현재/가장 최근 실패한 태스크
   - `/skip m1-s2-t3` → 특정 태스크

2. 의존성 영향 분석:
   - 이 태스크에 의존하는 후속 태스크가 있는지 확인
   - 있다면 경고:
     ```
     ⚠️ 이 태스크를 건너뛰면 다음 태스크도 영향받습니다:
     - m1-s2-t4 "충돌 감지" (이 태스크에 의존)
     
     계속 건너뛰시겠어요? [Y/n]
     ```

3. 승인 시:
   - 태스크 상태를 `skipped`로 변경
   - execution-state.json 갱신
   - 의존 태스크도 자동으로 `skipped` 또는 사용자 판단

4. "태스크 [이름]을 건너뛰었습니다. `/execute`로 다음 태스크를 진행하세요."
