# Cost — 비용 요약

프로젝트 실행에 사용된 비용을 모델별, 단계별로 요약한다.

## 사전 조건

- `_workspace/execution-state.json`

## 출력 형식

```
비용 요약 — 크롬 공룡게임 클론

모델별:
  haiku:  3회 실행,  $0.003  (2%)
  sonnet: 8회 실행,  $0.32   (78%)
  opus:   2회 실행,  $0.08   (20%)
  ─────────────────────────
  합계:   13회,      $0.403

단계별:
  마일스톤 1: $0.35 (12/12 태스크)
  마일스톤 2: $0.053 (1/8 태스크 — 진행중)
  마일스톤 3: 미시작

기타:
  PM (계획 수립):  $0.12
  검증:           $0.05
  ─────────────────────────
  총 비용:         $0.573
  
예상 남은 비용: ~$0.45 (남은 태스크 19개 기준)
예상 총 비용:   ~$1.02
```

## 데이터 수집

각 태스크의 `cost` 필드와 execution-state의 기록을 기반으로 집계한다.
PM 호출(plan, refresh, pivot)과 검증(validate) 비용도 별도로 추적한다.
